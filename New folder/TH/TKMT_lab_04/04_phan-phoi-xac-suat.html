Phân Phối Xác Suất (PPXS)¶

Xc Sut l i lng dng o kh nng xy ra ca mt s kin (event), xc sut c gi tr t 0 n 1. Khi lm vic vi tp d liu, chng ta thng c nhu cu tm hiu xem cc thuc tnh (trng) d liu c tun theo quy lut g hay khng, c th d on c gi tr ca thuc tnh hay khng, gi tr no xy ra nhiu, gi tr no xy ra t?... tr li cu hi ny chng ta c th da vo hnh dng phn phi ca thuc tnh d liu xem n c hnh dng no c bit hay khng? Trong thng k, cc nh khoa hc gi nh cc c trng d liu l cc bin ngu nhin v tm hiu kh nng xut hin cc gi tr ca chng. Khi nim Phn Phi Xc Xut (PPSX) th hin xc sut xut hin ca cc gi tr ca bin ngu nhin.

C rt nhiu PPSX c tm ra, mi loi c phn bit da vo tnh cht ca bin ngu nhin hoc php th. Khi hiu c tnh cht mt s loi PPSX ri, cc bn c th so snh xem thuc tnh d liu m mnh ang tm hiu c ging vi mt PPXS no hay khng?

Trong ni dung ca bi vit ny chng ta ch kho st mt s phn phi tiu biu.

Trong Thng K, c rt nhiu PPXS, mi PPXS c hnh dng th khc nhau dng m hnh ha cc i lng ngu nhin khc nhau. Trong ni dung ny chng ta s kho st mt s PPXS nh:

  • Phn phi Nh Thc
  • Phn phi Chun
